# תיעוד סימולטור ה-ISA

מסמך זה מתעד את מבנה הקוד והמימוש של הסימולטור. לכל זוג קבצים (C ו-H) מוקדש סעיף המתאר את תפקידו במערכת.

## מבנה קבצי הפרויקט

### simulator.h
(הגדרות גלובליות, מבני נתונים מרכזיים כמו `SimState` ו-`Instruction`, וקבועי מערכת)

### main.c
(נקודת הכניסה לתוכנית, ניהול מחזור החיים של הסימולציה וקריאה לפונקציות הניהול)

### core.c / core.h
(מימוש ליבת המעבד: פענוח הוראות, ביצוע ה-Fetch-Decode-Execute וניהול הרגיסטרים)

### memory.c / memory.h
(ניהול הזיכרון הראשי של המערכת, פונקציות קריאה וכתיבה)

### disk.c / disk.h
(סימולציה של רכיב הדיסק, ניהול פקודות DMA ותזמון פעולות קלט/פלט)

### timer.c / timer.h
(ניהול רכיב הטיימר והפסקות (Interrupts) הנובעות ממנו)

### monitor.c / monitor.h
(ניהול תצוגת המסך (Monitor) והזיכרון הייעודי שלה)

### files.c / files.h
(ניהול קבצי הקלט והפלט: קריאת `memin.txt`, כתיבת קבצי ה-Trace והפקת תוצרי הסיום)
